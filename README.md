# sirpackages

> vue组件库

> 不断维护版本迭代中

### 项目特点
1. 参考现在社区比较成熟如ElementUI，Ant Design，Material UI这些UI轮子的API设计和样式布局。
2. 该项目基于Vue2.x版本来开发和使用

## Build Setup

``` bash
# install dependencies
npm install sirpackages

```
### 支持组件（持续更新研发中。。。。）
1. contain布局
2. card卡片
3. navmenu导航菜单
4. scrolltop滚动条
5. tabs选项卡
